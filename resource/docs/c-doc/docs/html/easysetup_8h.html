typedef void(* ESReadUserdataCb) (OCRepPayload *payload, char *resourceType, void **userdata)

A function pointer for registering a user-defined function to parse user-specific properties from received POST request.

Parameters
payloadRepresents a received POST request. If you know user-specific property key, then you can extract a corresponding value if it exists.
resourceTypeUsed to distinguish which resource the received property belongs to
userdataUser-specific data you want to deliver to desired users, i.e. application. The user should know a data structure of passed userdata.
typedef void(* ESWriteUserdataCb) (OCRepPayload *payload, char *resourceType)

A function pointer for registering a user-defined function to set user-specific properties to a response going back to a client.

Parameters
payloadRepresents a response. You can set a specific value with specific property key to the payload. If a client receives the response and know the property key, then it can extract the value.
resourceTypeUsed to distinguish which resource the received property belongs to.

This function Initializes the EasySetup.

This API must be called prior to invoking any other API

Parameters
isSecuredTrue if the Enrollee is operating in secured mode.
resourceMaskProvisining Resource Type which application wants to make. ES_WIFICONF_RESOURCE = 0x01, ES_COAPCLOUDCONF_RESOURCE = 0x02, ES_DEVCONF_RESOURCE = 0x04
callbacksESProvisioningCallbacks for updating Easy setup Resources' data to the application
Returns
ES_OK on success, some other value upon failure.
ESResult ESSetCallbackForUserdata ( ESReadUserdataCb  readCb,
ESWriteUserdataCb  writeCb 
)

This function is to set two function pointer to handle user-specific properties in in-comming POST request and to out-going response for GET or POST request.

If you register certain functions with this API, you have to handle OCRepPayload structure to set and get properties you want.

Parameters
readCba callback for parsing properties from POST request
writeCba callback for putting properties to a response to be sent
Returns
ES_OK on success, some other value upon failure.
